# git commit 템플릿이란
# git commit -m "" 에서
#  "" 안에 들어갈 내용의 형식을 정해놓는 것
# git commit -m "커밋 메시지"
# 와 동일하게 작용

작성 방법
1. git add <파일명> 입력
2. git commit 입력
3. insert를 위한 키보드 i 버튼 누르기
4. commit message 작성 끝나면 esc로 종료
5. shift : 템플릿에서 빠져나온 후 wq 입력 그리고 엔터 

의문점 :
git commit template의 내용을 수정하면 git repo에 push된 이후에 
수정된 내용으로 출력될 것으로 예상했으나,
바로 수정된 내용으로 출력되었음
